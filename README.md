# Estate Management System

A full-stack web application built for managing residential estate operations.

> ⚠️ Source code is private and owned by the client.

## Overview
This system was designed to help estate managers handle tenant records,
complaints, work orders, and administrative reporting from a single dashboard.

## My Role
- Designed the system architecture
- Built the frontend dashboard
- Implemented backend APIs and authentication
- Worked with real-world estate operational requirements

## Tech Stack
- React
- Node.js
- Express
- MongoDB
